# HouseLoop Backend

Proyecto backend de HouseLoop desarrollado con Node.js, Express y MySQL.

## Repositorio GitHub
https://github.com/Juanse01355/houseloop-backend

## Tecnologías
- Node.js
- Express
- MySQL2
- XAMPP

## CRUD implementado
- Create → POST /propiedades
- Read → GET /propiedades
- Update → PUT /propiedades/:id
- Delete → DELETE /propiedades/:id
